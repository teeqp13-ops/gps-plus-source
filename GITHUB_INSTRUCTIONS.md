# دليل رفع Gps Plus إلى GitHub Pages (محدث)

لقد تمت إضافة ميزة **إشعار جلب الشهادات من الخادم** داخل ملف السورس. اتبع الخطوات التالية لرفع الملفات:

## 1. إنشاء مستودع (Repository) جديد
1. اذهب إلى [GitHub](https://github.com/) وسجل دخولك.
2. اضغط على **New** لإنشاء مستودع جديد.
3. سمِّ المستودع (مثلاً: `gps-plus-source`).
4. اجعله **Public** (عام).
5. اضغط على **Create repository**.

## 2. رفع الملفات
1. داخل المستودع، اضغط على **uploading an existing file**.
2. ارفع الملفات التالية:
   - `GpsPlus.ipa` (الملف الأصلي لديك)
   - `source.json` (الملف المحدث المرفق هنا)
   - `index.html`
3. اضغط على **Commit changes**.

## 3. تفعيل GitHub Pages
1. اذهب إلى **Settings** > **Pages**.
2. اختر الفرع `main` والمجلد `/ (root)`.
3. اضغط على **Save**.

## 4. تحديث الروابط
بعد الرفع، ستحصل على رابط مثل: `https://[username].github.io/gps-plus-source/`
- افتح ملف `source.json` على GitHub.
- قم بتغيير `username` إلى اسم حسابك في GitHub في كل الروابط الموجودة داخل الملف.
- احفظ التغييرات.

**ملاحظة:** الإشعار سيظهر للمستخدمين عند تحديث السورس في التطبيقات المدعومة.
